# Geslaagd Checker

Statische website waarmee leerlingen voor havo, atheneum en gymnasium hun profiel, keuzevak, SE-cijfers en CE-cijfers kunnen invullen om te controleren of ze geslaagd zijn.

## Online zetten

Upload deze bestanden naar een statische host zoals Netlify, Vercel, GitHub Pages of een gewone webserver:

- `index.html`
- `styles.css`
- `script.js`
- `favicon.svg`
- `site.webmanifest`
- `robots.txt`

Er is geen backend, database of build-stap nodig. De site werkt direct zodra `index.html` online staat.

## Lokaal testen

Open `index.html` in een browser. Alles draait volledig in de browser.
